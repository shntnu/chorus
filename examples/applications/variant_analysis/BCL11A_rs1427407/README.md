# BCL11A Enhancer Variant — Fetal Hemoglobin / Sickle Cell

## Variant: rs1427407 (chr2:60,490,908 G>T)

This variant sits in the **BCL11A +62 DHS enhancer** — the single most
significant GWAS SNP for fetal hemoglobin (HbF) levels (P = 7.23e-50).
It disrupts a GATA1/TAL1 composite motif in erythroid progenitors.

The BCL11A enhancer is the therapeutic target of **Casgevy** (Vertex/CRISPR),
the first FDA-approved CRISPR gene therapy for sickle cell disease. The
therapy disrupts the nearby +58 DHS element to reactivate fetal hemoglobin.

## Tracks

Predicted with AlphaGenome using K562 (erythroleukemia) tracks:
- DNASE, H3K27ac, CAGE (+/-), RNA-seq (+/-)

**Note on cell-type specificity**: The BCL11A enhancer is most active in
primary erythroid progenitors. K562 shows modest effects because this cell
line has a different chromatin state at this locus. Primary CD34+ erythroid
differentiation tracks would show stronger effects.

## Key findings

- Modest chromatin accessibility change (-0.113 log2FC in K562)
- Minimal expression effects — consistent with K562 not being the optimal
  cell type for this enhancer
- Demonstrates importance of cell-type selection for variant interpretation
